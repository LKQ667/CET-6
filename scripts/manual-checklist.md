# CET-6 二期手工验收清单（20+）

执行前置条件：

1. 本地已启动：`npm run dev`
2. 浏览器打开：`http://localhost:3000`
3. 如需云端模式，已配置 Supabase 环境变量并执行 `schema.sql` + `seed.sql`

---

## A. 登录与跨端同步

- [ ] A1 输入邮箱后可发送验证码
- [ ] A2 mock 模式输入 `123456` 可以登录
- [ ] A3 登录后页面自动拉取今日任务、词汇、提醒设置、游戏档案
- [ ] A4 刷新页面后登录态仍保留（localStorage）
- [ ] A5 退出登录后恢复游客视图

## B. 今日副本与 RPG 反馈

- [ ] B1 当日默认出现 4 个任务（词汇/听力/写译/阅读）
- [ ] B2 点击“开战并打卡”后任务状态变为“已击败”
- [ ] B3 完成任务后战斗播报显示经验/金币/等级/连胜信息
- [ ] B4 等级经验条会随经验变化
- [ ] B5 周六能显示 Boss 卡；非周六显示“周六开启”提示
- [ ] B6 周六清空任务后 Boss 状态可变为“已击败”

## C. 词汇真实性与出处追溯

- [ ] C1 今日词汇可展示词条列表
- [ ] C2 点击词条后显示年份/月/套卷/题型/来源片段
- [ ] C3 有来源链接可点击打开
- [ ] C4 若带 `source_file`，详情中可看到来源文件名

## D. 官方入口与内置资料

- [ ] D1 资源中心存在官方入口 `https://cet.neea.edu.cn/`
- [ ] D2 `cet4-6.pdf` 可下载
- [ ] D3 `大学英语六级词汇乱序版.pdf` 可下载

## E. 上传抽词流程

- [ ] E1 上传 TXT/PDF 成功
- [ ] E2 上传后可触发抽词
- [ ] E3 页面显示候选词数量（待审核）
- [ ] E4 候选词默认不直接进入 verified 主库

## F. 提醒系统

- [ ] F1 邮件提醒开关可切换并保存
- [ ] F2 浏览器推送开关可切换并保存
- [ ] F3 点击“开启浏览器推送”后可成功订阅（支持的浏览器）
- [ ] F4 `POST /api/reminder/cron-send` 可执行，且同一时段不重复发送

## G. 响应式与体验
